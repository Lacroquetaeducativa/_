How To Customize Main Menu Links?:

To add links in main menu navigation do as mentioned below:

    1. Go to Blogger >> Template >> Edit HTML
    2. In the template, search for <!--Main Menu-->
    3. Now you will find the whole of the menu, customize it as you like.
    4. One everything is done, Save the template.
